## ⚙️ Как запустить проект

### 1. Клонировать репозиторий
```bash
git clone https://github.com/TimonMax/nginx_lab_4.git
cd nginx_lab_4
```
### 2. Запустить контейнеры Docker
```bash
docker-compose up -d --build
```
### 3. Открыть в браузере
```bash
http://localhost:8080
```
### 4. Проверка работы PHP
```bash
http://localhost:8080/phpinfo.php
```
## Содержимое проекта
```docker-compose.yml``` — описание сервиса Nginx

```www/form.html``` — главная HTML-страница с формой

```www/index.php``` — подключает form.html

```www/phpinfo.php``` — файл с phpinfo() для проверки

```www/process.php``` — серверный обработчик: валидация, сессия, запись в data.txt

```www/view.php``` — страница для просмотра всех сохранённых записей (читает data.txt)

```www/data.txt``` — файл с сохранёнными записями (создаётся автоматически при первой записи)

```nginx/default.conf``` — файл для обработки PHP

```www/UserInfo.php``` — информация о пользователе

```www/ApiClient.php``` — работа с Api
